ReWOO Ilość zapytań: 2-3 wywołania LLM

Reasoning WithOut Observation — model językowy tworzy kompletny plan realizacji zadania przed rozpoczęciem pracy. Następnie jest w stanie wywołać poszczególne narzędzia równolegle, aby na koniec połączyć otrzymane wyniki.

  ┌──────────────────────────────────────────────────────────┐
  │  Faza 1: PLANNER (LLM)                                   │
  │                                                          │
  │  User: "Porównaj Python i Rust pod kątem wydajności."    │
  │            │                                             │
  │            ▼                                             │
  │  Plan: 1. search("Python performance")                   │
  │        2. search("Rust performance")                     │
  │        3. Combine results                                │
  └──────────────────────────┬───────────────────────────────┘
                             │
         ┌───────────────────▼─────────────────────┐
         │  Faza 2: WORKER (równolegle)            │
         │                                         │
         │  ┌──────────────┐  ┌──────────────┐     │
         │  │ search       │  │ search       │     │
         │  │ "Python..."  │  │ "Rust..."    │     │
         │  └──────┬───────┘  └──────┬───────┘     │
         │         └────────┬────────┘             │
         └──────────────────┼──────────────────────┘
                            │
         ┌──────────────────▼───────────────────────┐
         │  Faza 3: SOLVER (LLM)                    │
         │  Łączy obserwacje → ostateczna odpowiedź │
         └──────────────────────────────────────────┘

Zalety

  • Mniej wywołań LLM — tylko 2-3 (planner + solver)
  • Równoległość — narzędzia działają jednocześnie
  • Niższy koszt — mniej wywołań -> mniej tokenów

Wady

  • Brak adaptacji — plan jest stały
  • Jakość planu = jakość wyniku
  • Wymaga modelu o wyższej mocy i jakości

Najlepszy gdy kroki są przewidywalne i niezależne, na przykład: "porównaj X i Y", "zbierz dane o A, B, C". Nie nadaje się do zadań eksploracyjnych i researchowych.
