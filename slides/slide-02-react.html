ReAct (Reasoning + Acting)

Agent przeplata myslenie i dzialanie w petli. Kazdy krok: model rozumuje, wykonuje akcje, obserwuje wynik, i rozumuje ponownie.

User: "Kim byl Mikolaj Kopernik?"

┌── LLM ────────────────────────────────────────────┐
│ Thought: I need to search Wikipedia for Copernicus  │
│ Action: wiki_search("Mikolaj Kopernik")             │
└────────────────────────────────────────────────────┘
                      │
                      ▼
              ┌───────────┐
              │ Tool: search │ ──→ returns Wikipedia article
              └───────────┘
┌── LLM ───────────────────────────────────────────┐
│ Observation: [Wikipedia content about Copernicus]   │
│ Thought: I now have enough information              │
│ Final Answer: Mikolaj Kopernik byl...               │
└────────────────────────────────────────────────────┘

Zalety

  • Transparentny — kazdy krok rozumowania widoczny
  • Elastyczny — model adaptuje plan do wyników
  • Dziala z modelami tekstowymi (jak Bielik 7B)

Wady

  • Token-heavy — pelna historia w kazdym zapytaniu
  • Moze zapetlic sie bez Final Answer
  • Sekwencyjny — narzedzia jedno po drugim
workflow:
  _type: react_agent
  tool_names: [wiki_search, current_datetime]
  llm_name: bielik
  max_tool_calls: 5
