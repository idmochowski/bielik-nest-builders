Czym jest Agent AI?

Agent AI to połączenie modelu językowego (LLM) oraz narzędzi (funkcji), które model może wykorzystać do realizacji zadań.

Agent = model LLM + harness. Mianem 'harness' określamy każdy fragment kodu, narzędzi czy konfiguracji, który nie jest modelem językowym. To sposób na dostosowanie modelu do konkretnych zadań i realizacji celów.
